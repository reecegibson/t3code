# T3 Code — reecegibson fork

> A minimal web GUI for coding agents, forked from [pingdotgg/t3code](https://github.com/pingdotgg/t3code) with extra provider support and UX improvements.

The upstream project is Codex-only. This fork extends it with **Claude Code as a first-class provider** and a handful of quality-of-life UI tweaks that make the experience feel less rigid.

## What's different in this fork

| Feature | Description |
|---|---|
| **Claude Code provider** | Run Claude Code sessions alongside Codex — same UI, new brain |
| **Resizable sidebar** | Drag the sidebar edge to whatever width you want instead of being stuck with a fixed layout |

More coming soon. This is actively being hacked on.

## Getting started

> [!WARNING]
> You need [Codex CLI](https://github.com/openai/codex) installed and authorized for the Codex provider to work.

### Quick start

```bash
npx t3
```

### Desktop app

Grab the latest build from the [Releases page](https://github.com/pingdotgg/t3code/releases).
